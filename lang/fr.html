Ordinateur de Bureau

Potabi pour Ordinateur de Bureau

Nous travaillons sur un systèmes d'exploitation des ordinateurs de bureau. C'est construit sur le même fondement comme tout le reste que le Potabi Systems projet. Cela peut sembler "de base", nous allons encore plus loin. Bien que chaque système d'exploitation soit différent, ils sont intentionnellement conçus pour leurs plateformes. C'est moins courant. De nombreux serveurs Linux tombent souvent dans quelques problèmes simples. Soit ils peuvent être axé sur Systèmes d'exploitation informatiques pour ordinateur de bureau, et les serveurs sont des versions légèrement modifié de leur Systèmes d'exploitation informatiques pour ordinateur de bureau sans trop changer. Alternativement, ils sont conçus pour les serveurs, et leur Systèmes d'exploitation informatiques pour ordinateur de bureau a de légers changements pour le bureau. Ce n'est pas nous. Potabi Desktop l'édition est CONÇUE pour le bureau.

Potabi pour Serveurs

Tel que ordinateur de bureau et téléphone portable, Potabi Server l'édition est conçue pour l'utilisation du serveur. Nous avons coutume applications de programme pour vous aider à mettre en place des outils pour hébergement de sites Web, stockage de données, et plus. Il ne vient pas avec un logiciel qu'aucun système d'exploitation de serveur aurait jamais besoin, comme un navigateur Web, jeux vidéo, un graphique magasin de logiciels, logiciel de caméra, lecteur multimédia, etc. Lors de l'utilisation d'un serveur, il doit être mince, il doit être vite, il doit être utilisable. Potabi Server ne prend pas plus de place qu'il n'en faut pour que ce soit une expérience conviviale. Soyez opérationnel aussi vite que possible avec Potabi Server.

Potabi pour Téléphone Portable

Potabi Mobile l'édition est le premier FreeBSD projet de annoncer officiellement commencer les travaux sur étant basé sur FreeBSD système d'exploitation de téléphone portable être à la hauteur, et avec le même fondement téléphone portable environnements, avec les nouvelles variantes de Linux. Utilisons Lomiri la page des téléchargements ne vous donne que les téléphones mobiles pris en charge. Nous travaillons pour nous assurer Android, avec un logiciel personnalisé, fonctionne parfaitement, et avec le moins de problèmes possible. L'édition Potabi Mobile a été la façon dont le Potabi Systems projet a commencé en premier lieu.
