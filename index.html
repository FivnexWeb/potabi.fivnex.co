Our mission

is to bring higher security, and an easier experience, to desktop and mobile computing. Based off of FreeBSD, with emphasis on simplicity for our user experience and design, Potabi is an excellent vendor for open-source desktop and mobile operating systems.

The Potabi Systems project is a FreeBSD-based operating system that works to make a fantastic desktop operating system experience, based around mostly permissive licensed software. The project is also working on it's sister project CoreNGS, which is a fork from FreeBSD with the goal of being more independent. Potabi Systems believes independence and self-sustainability is an important aspect of software. Our developers are mostly inspired by Linux Mint and Pop!_OS. The Potabi Systems project aims to be safe without being intrusive. To be smart without being complicated. To be interesting without being trendy. To be simple without being vague.
