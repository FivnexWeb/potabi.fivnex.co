About Potabi:

  • * Pronounced "P-oh-t-hah-bee"
  • * The logo margins are 1/3x on all sides or above, just make it look nice and give the logo room to breathe.
  • * The logo should be placed where it is clear and visible
  • * The logo for the Potabi Systems project is a dragon named "Feaye", she/her pronouns.
  • * We do ask you respect our logo's pronouns.
  • * Yes, there is officially lore to the logos.
  • * Codenames for Potabi Systems are generally names of friends and family of developers of the project.
  • * Point releases (i.e x.1, x.2) are modifications to that release, and changes aren't usually major.
  • * Potabi Systems is a stable-release, not rolling-release.
  • * We release when the software is ready, not when some arbitrary deadline hits.
  • * Potabi release versions simply count up from 1, because complex version numbers are stupid.
  • * Some Potabi releases come with a postfix, this can include
    • > * A - Alpha/testing release
    • > * E - Experimental release
    • > * U - Updated software release (Not much in changes, just updated software)
